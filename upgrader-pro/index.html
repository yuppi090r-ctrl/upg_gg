03 — Механика

Как работает
апгрейд

На одном слайдере строится весь продукт. Простота и есть главный актив.

Шаг за шагом

  • Выбор скина из Steam-инвентаря или баланса
  • Выбор целевого скина из каталога (дороже)
  • Система показывает точный % шанса ДО подтверждения
  • Шанс зависит от соотношения стоимости
  • Быстрые кнопки множителей или шансов
  • Слайдер для точной настройки
  • Кнопка «Прокачать» — результат мгновенный
  • При проигрыше — кэшбэк-кейс

Provably Fair

  • Каждый апгрейд — собственный хеш
  • SHA-256 от Server Seed публикуется заранее
  • Игрок задаёт Client Seed
  • Результат = Server Seed + Client Seed + Nonce
  • После серии — можно проверить хеш
  • Plus: технически шанс не подкручивается в моменте

Три кнопки шанса (35/55/75%) выбраны не случайно: это не слишком безопасно, чтобы утомлять, но и не слишком рискованно, чтобы отпугнуть новичка. Математически — 55% это sweet spot где дофамин выигрыша ощущается каждые ~1.8 попытки. Классический Skinner box с variable-ratio подкреплением.

04 — RTP и House Edge

«До 99%» —
правда и миф

Заявленный RTP 99% — это честная правда про одну конкретную игру. Вся воронка работает иначе.

Что говорят публично

  • RTP «до 99%» на главной странице
  • «Самая честная апгрейд-платформа»
  • Provably Fair как доказательство
  • «Without wagering!» в маркетинге
  • VIP-кэшбэк 5–15% как «возврат»

Что по математике

  • RTP 99% =правда только для x1.5 без бонусов
  • На x4–x8: эффективный edge 5–10%
  • Средневзвешенный по всем ставкам: 4–7%
  • По оценкам tested.gg: 40–60% (агрессивно)
  • Индустриальный стандарт: 2–10%
  • Скрытый edge #2: мид-прайс спред 10–15%

Реальный RTP по всей воронке: 55–75%

После вагеров на бонусах (x2–x5 оборота), кэшбэк-циклов (с вагером x3), мид-прайс спредов на целевых скинах (10–15% дороже рынка) и wager-wall перед выводом — реальный эффективный возврат 55–75%. Остальные 25–45% — маржа платформы.

  • Пример: юзер с депозитом $5 делает серию из 8 апгрейдов x2-x4
  • При edge 5% и обороте $40 (×8 прокрутов) ожидаемая потеря = $2 = 40% депозита за вечер
  • Даже при «выигрыше на дистанции» — кэшбэк и wager съедят 85% бонуса

Сильные стороны

Фокусировка на одной механике

Только апгрейды. Нет кейсов, crash, рулетки. Это и главный недостаток (меньший ассортимент), и главное преимущество (чистый UX).

Provably Fair из коробки

Не пытается скрыть математику. Каждый апгрейд верифицируем через хеш. В нише, где CSGORoll обнулял балансы игрокам, это серьёзное отличие.

Низкий порог входа

Минимум ~$0.5 = массовый funnel для подростков с карманными. Компромисс: underage-риск, но огромный top-of-funnel.

СНГ-first маркетинг

Telegram 90.5K, Pikabu SEO-сетка, VK, блогеры на 100% компенсации. Каналы стоят копейки, ROI x5–x20.

Слабые места

Юридика «серая»

Три юрлица без игровой лицензии. Один ASA/FCA запрос или EU-бан лутбоксов — и маркетинговые каналы умирают. Нет Curacao/Anjouan-wrapper для выхода в западные маркеты.

Underage-мина

KYC запрашивают только при крупных выводах. Школьники 13-15 депозитят с карт родителей. Один громкий случай — congressional hearings или UK review v2.

Вывод только скинами

Пользователь заперт в экосистеме. Для кэша — вторичная продажа со своей комиссией 10–45%. Снижает лояльность: юзер чувствует что его «не выпускают».

Trustpilot 2.9/5

62% одна звезда. Сложно масштабировать на Запад, где доверие к платформе считается в Trustpilot-звёздах. Структурно сложно выправить — нужна публичная работа саппорта.

Valve-риск

Обвал 23.10.2025 (trade-up контракт) = -48% рынка за 48 часов. Upgrader.pro целиком зависит от цен на Steam Market, любой апдейт Valve рушит инвентари и маржу.

11 — Выводы

Что брать
в свой продукт

Upgrader.pro — живой, рабочий, прибыльный. Но построен на шатком фундаменте. Для нового продукта в нише есть конкретные уроки.

Что брать

  • Апгрейд с явным % шанса ДО — рабочая механика, прозрачнее лутбоксов
  • Provably Fair с первого дня — SHA-256, Server Seed + Client Seed
  • Steam OAuth + депозит скинами — super-low friction
  • Telegram-first для СНГ-аудитории — лучший ROI канал
  • Промокоды через партнёров — бесплатный CAC, чистая exchange-экономика
  • 4-tier VIP с прогресс-баром — sunk cost удерживает китов
  • Кэшбэк по пятницам — habit loop работает
  • Фокус на одной механике — не пытаться быть «всеми казино сразу»

Что делать иначе

  • Прозрачный take rate, а не маскированный «RTP до 99%»
  • KYC + age-gate с депозитов от $100 — снять underage-риск
  • Fiat-вывод (крипта/PayPal) в дополнение к скинам — доверие
  • Игровая лицензия (Anjouan/Curacao) с первого дня — legit-wrapper
  • Открытый Trustpilot-саппорт — отвечать на каждый одно-звёздочный
  • Нет fake winning videos — блогерам не 100% компенсация
  • Вагер ≤ x1 на всех бонусах — иначе щедрость превращается в ловушку
  • Публичный P&L или хотя бы «открытая математика» — новый quadrant доверия
Главная мысль

Upgrader.pro — это работающая модель с шатким фундаментом

Механика кристально понятная. Экономика сходится — house edge 4–7% × оборот ×8 = 40% депозита за сессию. Retention-петли отстроены: daily, кэшбэк, VIP, реактивация. Маркетинг на бесплатных партнёрах даёт ROI x5–x20.

Но фундамент хрупкий: три юрлица без лицензии, 62% одно-звёздочных на Trustpilot, underage-аудитория, полная зависимость от Valve-цен. Один регуляторный удар (EU-бан лутбоксов, US congressional hearings) — и вся структура теряет маркетинговые каналы и процессинг.
