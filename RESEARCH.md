# UPG_GG — Исследование ниши

## Референсы

### Upgrader (upg.gg → upgrader.pro)
- **Тип:** гэмблинг — апгрейд скинов CS2 с шансом
- **Механика:** закидываешь скин/баланс, выбираешь предмет дороже, шанс 4-80% (множители x2/x4/x8 или фиксированные 35%/55%/75%)
- **Авторизация:** Steam
- **Онлайн:** ~1500, 57М+ апгрейдов всего
- **Юрлица:** LLC «Transtrade» (Бишкек, Кыргызстан), ITSFAIL OÜ (Таллин, Эстония)
- **Соцсети:** VK, Telegram
- **Фишки:** лайв-лента апгрейдов, ивенты (Easter Event), розыгрыши, push-уведомления
- **Монетизация:** математическое преимущество при апгрейде (house edge)

### SkinSwap (skinswap.com)
- **Тип:** маркетплейс — обмен, покупка, продажа скинов
- **Игры:** CS2 + Rust
- **Механика:** обмен скин-на-скин через бота, продажа за реальные деньги, покупка с маркета
- **Выплаты:** Bitcoin, Ethereum, Litecoin, PayPal
- **Цифры:** 480K обменов, 500K продаж, 680K юзеров
- **Trustpilot:** 3.8/5 (1751 отзыв)
- **Юрлицо:** DES Labs, LLC
- **Стек:** Next.js, Intercom (поддержка), Trustpilot виджеты
- **Фишки:** 40% бонус на депозит, бесплатный предмет до $50, партнёрка, блог, раздел Китай
- **Монетизация:** комиссия с обменов/продаж

## Сравнение моделей

| | Upgrader | SkinSwap |
|---|---|---|
| Тип | Гэмблинг | Маркетплейс |
| Игры | CS2 | CS2 + Rust |
| Монетизация | House edge | Комиссия |
| Вывод денег | Нет | Да (крипта, PayPal) |
| Аудитория | Рисковые геймеры | Трейдеры |
| Юрисдикция | Кыргызстан + Эстония | США (LLC) |
